# Antigravity (수학·과학 주제탐구 보조 프로그램)

한국 고등학교 1~3학년 학생들이 2022 개정 교육과정에 맞춘 수학·과학 주제탐구를 주도적으로 수행하고 학술 규격에 맞춘 고품질 보고서를 완성할 수 있도록 지원하는 8단계 가이드형 웹 애플리케이션입니다.

---

## 🌟 핵심 차별화 포인트

1. **명시적인 가설 설정 단계 (4단계)**: 기존 제품에는 없는 구조화된 가설과 논리적인 근거 도출을 유도합니다.
2. **탐구 유형 5종 분기**: 실험탐구, 데이터·통계, 수학적 모델링, 조사연구, 문헌사례 유형에 따라 동적으로 입력 필드와 AI의 가이드 멘토가 변경됩니다.
3. **가설 및 변인 연동 파이프라인**: 4단계에서 지정한 가설이 5, 6, 7단계의 prefill 및 리드온리 카드로 유기적으로 흐르며 시제 전환(미래 계획 -> 과거 수행 완료)까지 자동 연계됩니다.
4. **교과 연계 자기점검 모달**: 5단계 끝에서 교과 단원 성취기준과의 융합도를 등급(✓, △, •) 형태로 점검해 주는 메타인지 학습 자가진단이 가동됩니다. (강제 차단이 없으며 주도권을 학생이 쥡니다.)
5. **윤리적 가드레일 엄격 준수**:
   - 학생용 UI 전체에 **"세특"**, **"생기부"** 등의 교육을 왜곡할 수 있는 키워드 차단.
   - AI 답변 추천은 '제안'일 뿐, 학생이 **반드시 스스로 가공 및 확정**하게 유도하는 학습 중심 설계.
   - 자기점검 모달 내에 정량적 수치 평가 및 별점(★) 표시 배제.
6. **A4 최적화 인쇄 양식**: 브라우저 자체의 초고화질 벡터 인쇄 엔진(`window.print()`)과 매치된 미려한 학술 양식의 A4 프린트 스타일 내장.

---

## 📂 파일 구조

- `index.html`: 단계별 입력 폼 구조와 우측 실시간 AI 멘토/도움말 패널로 나뉜 프리미엄 SPA 뼈대.
- `style.css`: 짙은 청록(Primary)과 따뜻한 주황(Accent)을 테마로 한 초현대적 사이버네틱 학술 테마(글래스모피즘, 글로우 모서리, 유려한 3D 트랜지션 애니메이션) 및 A4 출력용 `@media print` 커스텀 스타일시트.
- `data.js`: 2022 개정 교육과정 32과목 대분류 정보, 2015 개정 과목 정보, 5대 탐구 유형 및 변수 템플릿.
- `mock-ai.js`: 교과목 및 단어를 분석하여 맞춤 주제 5종 제안, 각 텍스트 필드별 3종의 지능적 추천, 가설 타당성 채점, 교과 연계 메타인지 피드백을 실시간 연산하는 스마트 Mock AI 엔진.
- `pdf-export.js`: 전 단계 데이터를 학술 논문 스타일의 A4 프리뷰 모달로 구축하고 인쇄 및 JSON 내보내기를 담당하는 출력 제어기.
- `app.js`: `localStorage` 30초 자동 저장 바인딩, 단계별 폼 검증, 시제 전환 및 prefill 가동, 내비게이션 바인딩 등 핵심 비즈니스 로직.

## 🚀 실행 및 로컬 구동 방법

이 프로젝트는 무거운 패키지나 서버 환경 세팅이 필요 없는 **클라이언트 사이드 모듈 구조**를 지원합니다. 아래의 두 가지 방법 중 원하는 형태로 자유롭게 실행할 수 있습니다.

### 방법 1: 즉시 실행 (Zero-Setup)
1. 다운로드한 `antigravity-inquiry-helper` 폴더를 엽니다.
2. `index.html` 파일을 크롬, 엣지, 사파리 등의 **브라우저로 더블클릭**하여 즉시 실행합니다.
3. 추가 설치 없이 즉각적이고 부드러운 글래스모피즘 화면을 볼 수 있습니다.

### 방법 2: 로컬 개발 서버 실행 (Vite 기반)
코드 자동 리로드 및 핫 모듈 대체(HMR) 기능이 지원되는 로컬 서버를 가동하고자 할 때 사용합니다.
```bash
# 1. 프로젝트 폴더로 이동
cd C:\Users\user\.gemini\antigravity\scratch\antigravity-inquiry-helper

# 2. 관련 개발 패키지(Vite) 설치
npm install

# 3. 로컬 개발 서버 실행
npm run dev
```
터미널에 로드된 `http://localhost:5173` 주소를 브라우저로 띄워 로컬 서버 형태로 구동할 수 있습니다.
