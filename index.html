1. 탐구 기본정보 입력

학년 및 진로 분야, 그리고 탐구하고자 하는 핵심 교과목을 세팅합니다.

학습자 프로필 (진로 연계)
탐구 대상 교과 영역
⚠️ 안내 문구 영역

2. 탐구 주제 및 키워드 탐색

흥미 키워드를 입력하고 AI 주제 제안을 바탕으로 나만의 주제와 탐구 유형을 확정합니다.

Phase A: 관심 키워드 및 배경

탐구의 핵심이 될 과학 법칙, 용어 등을 입력하세요. 직접 입력하기 어렵다면 위의 진로 맞춤 키워드를 클릭해 보세요.

Phase B: AI 주제 후보 제안

원하는 주제를 클릭하면 하단 최종 주제창에 자동 입력됩니다.

Phase C: 최종 주제 및 탐구 유형
⚠️ 안내 문구 영역

3. 탐구 동기, 목적 및 핵심 질문

탐구를 구체화하기 위해 깊은 동기와 목표, 해결해야 할 중심 질문을 작성합니다.

📝 질문 단계입니다. 추측에 따른 결론(답)은 다음 4단계 가설 설정에서 구체화합니다.

4. 가설 설정 및 변인 구조화

핵심 질문에 대한 정량적이고 검증 가능한 가설 명제를 세우고, 조건 변인들을 슬롯에 배치합니다.

검증 가능하도록 "~한다면, ~할 것이다" 형식을 권장합니다.

선택한 탐구 유형별 핵심 변수 구조화

📊 AI 가설 입체 평가 진단서

5. 탐구 방법 및 절차 설계

가설 검증을 위해 설계 단계를 명확하게 논리 구조로 상세화합니다.

6. 탐구 수행 및 데이터 수집·분석

설계한 계획을 바탕으로 실제 탐구를 실행하고 모인 로우 데이터를 체계적으로 기록합니다.

5단계 계획 연계 prefill 아래는 5단계 방법 설계를 바탕으로 수행을 시작하기 위해 prefill된 시제 전환 결과물입니다. 자유롭게 수정해 보세요.
📊 [수집한 데이터로 쉽고 빠르게 통계 분석하기]

여러분이 관찰하거나 수집한 데이터(CSV 파일)가 있다면, 쉬운 데이터 분석 도구를 사용해 변수들이 서로 어떤 관계를 맺고 있는지 그래프와 정량 수치로 편리하게 알아볼 수 있습니다.

7. 탐구 결과 요약 및 가설 검증

모든 데이터와 관찰 사실을 정리하고, 초기에 수립했던 가설의 지지 여부를 명확히 판정합니다.

📈 [선택] 데이터 분석 결과 입력하기 (상관계수 및 유의확률)

앞의 '쉬운 데이터 분석기'에서 나온 결과 수치들을 여기에 적어주면, 최종 탐구 보고서에 멋진 통계 분석 표로 예쁘게 들어가고, AI 멘토가 가설 판정을 자동으로 도와줍니다.

2. 가설 검증 및 종합 결론

⚠️ AI는 결론을 대신 내려주지 않습니다. 학생이 모은 데이터에 입각하여 주도적으로 최종 문장을 작성해야 합니다.

8. 참고 문헌 및 자료 연동

데이터 신뢰성을 다지기 위해 실제 데이터베이스를 모사한 참고자료 검색 연동 및 교과서를 입력합니다.

학술 데이터베이스 모의 검색

💡 안내: 이 검색은 DBpia, RISS, KISS, KOSIS 등 실제 공인 학술 사이트 검색을 결합한 모의 체험 검색기입니다. 주요 학술 키워드(예: 물리, 화학, 생명, 수열, 미세먼지, 수면 등)를 입력해 난이도 등급별 추천 자료를 추가해 보세요! 실제 여러분이 찾은 다른 참고문헌들은 하단의 [수동 입력 추가]로 자유롭게 작성할 수 있습니다.

학술 키워드를 검색하면 관련 학술지 및 통계 기관 참고 자료가 로드됩니다.

교과 단원 매핑 진단 내역:
⚙️ Antigravity 시스템 설정

실시간 AI 탐구 설계를 사용하기 위한 설정을 관리합니다. 입력하신 API 키는 브라우저의 로컬 환경(LocalStorage)에만 안전하게 저장되며 외부로 수집되지 않습니다.

* Claude API는 브라우저 직접 호출 시 CORS 보안 에러가 나므로 우회 프록시 URL 주소를 입력해 주어야 작동합니다.

* 선택된 AI 공급자의 API 키를 비워둘 경우, AI 제안 시 모의 체험(Simulated) AI 모드로 자동 폴백하여 고품질의 어울리는 주제들을 설계합니다.

선생님이 제공한 공유 링크를 사용하지 않고 개별적으로 실시간 연동을 연결하려면 입력하세요. (비워둘 시 오프라인 모드로 자동 작동)

이미 계정이 있으신가요? 로그인
🤖 제미나이 웹 브릿지 (무료/API 키 없음)

학생의 개인 교육용 제미나이(gemini.google.com) 계정을 활용하는 안전한 연결 방식입니다.

  1. 아래 프롬프트 창 우측의 '복사하고 제미나이로 이동' 버튼을 클릭합니다.
  2. 열린 제미나이 웹페이지 채팅창에 붙여넣고(Ctrl+V) 전송합니다.
  3. 제미나이의 답변 전체를 복사(Ctrl+A 후 Ctrl+C 또는 복사 아이콘 클릭)합니다.
  4. 아래 '제미나이 답변 붙여넣기' 영역에 입력하고 '분석 및 적용하기'를 누르세요.
